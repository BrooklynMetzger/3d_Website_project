## 🧐 About <a name = "about"></a>

This is a small fun project I made to learn about Three.js, also inspired by learning the Met has an API.
This is a relexing little Art Gallery to walk around in and listen to music. 

## 🏁 Getting Started <a name = "getting_started"></a>

This website is hosted with Github pages, controls are listed in a tab on the menu.
